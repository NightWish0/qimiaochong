# qimiaochong

奇妙宠-宠物分享社交平台


使用技术:

* 后台
  * `Spring Boot` 快速搭建整个项目，去除繁琐的XML配置
  * `Spring`+`Spring MVC`+`MyBatis`框架
  * `Maven`构建项目
  * 页面引擎采用 `Thymeleaf`
  * 数据库连接池使用`Druid`
  * `Redis`集中式缓存
  * `Shiro`权限框架，权限基于RBAC设计
  * `Mapper`MyBatis通用扩展插件
  * `PageHelper`MyBatis分页插件
  * 数据库使用`MySQL`
* 前台
  * 后台模板采用基于[AdminLTE2](https://github.com/almasaeed2010/AdminLTE)
